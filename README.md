# Hospital_Manager

개요: C++ MFC를 기반으로 MySQL 데이터베이스와 FTP 서버를 연동하여, 회원의 신체 정보와 변화 과정전과 후를 시각적으로 추적할 수 있는 헬스케어/피트니스 맞춤형 데스크톱 애플리케이션

### 1. 개발 환경 및 기술 스택

- **Language / Framework:** C++ / MFC
- **Database:** MySQL
- **Network / Storage:** FTP Server (이미지 아카이빙), TCP/IP Socket (데이터 통신)
- **IDE:** Visual Studio 2022

### 2. 주요 기능 (Key Features)

- **CRUD 완벽 구현:** 회원 정보 저장, 조회, 수정, 삭제 기능.
- **스마트 클릭 이벤트:** * 리스트 `원클릭(Single Click)` 시: FTP 연동 없이 즉각적으로 로컬의 `first` 폴더 사진을 불러와 메인 화면에 렌더링 (잔상 제거 로직 적용).
    - 리스트 `더블클릭(Double Click)` 시: 촬영 일자 안내 팝업 출력 후, 모달 다이얼로그(`CGalleryDlg`)를 띄워 6분할 상세 갤러리 뷰 제공.
- **안전한 로컬 디렉토리 구축:** `CreateDirectory`를 활용해 기본 다운로드 경로, 카드번호, 날짜, First 폴더를 계층적으로 자동 생성.

### 3. 시스템 아키텍처 및 핵심 워크플로우

이 프로젝트는 3가지 핵심 모듈이 유기적으로 통신하며 작동합니다.

**① [Client UI] MFC 애플리케이션 (사용자 경험 및 렌더링)**

- 리스트 컨트롤(List Control)을 활용한 전체 회원 정보 직관적 조회.
- 사용자의 오입력을 방지하고 편의성을 높이는 데이터 자동 포맷팅 로직 (전화번호 하이픈 자동 삽입, 생년월일 유효성 검사).
- 고화질 이미지 렌더링 (`HALFTONE` 안티앨리어싱) 및 Picture Control을 활용한 UI 동적 업데이트.
- TCP/IP 스레드를 이용해 서버와 지속적으로 통신하며 데이터를 실시간으로 동기화.

**② [Database] MySQL (데이터 영속성 및 관리)**

- `customer` 테이블을 설계하여 회원의 고유 정보(카드번호, 이름, 생년월일, 키, 전화번호 등) 관리.
- 기존 회원 정보 수정(`UPDATE`)과 신규 회원 등록(`INSERT`) 로직을 분리.
- 신규 가입 시, 애플리케이션에서 자동으로 현재 시스템 날짜를 추출해 `reg_date`(접수일자)에 삽입하여 관리자의 수기 입력 최소화.

**③ [File Server] FTP 연동 (스마트 폴더링 및 이미지 관리)**

- 회원별 카드번호를 디렉토리명으로 사용하여 독립적인 파일 공간(`download\100\...`) 할당.
- **[킬러 기능: First 사진 영구 보존]** 해당 회원 폴더 내에 `first` 디렉토리가 존재하는지 윈도우 API(`GetFileAttributes`)로 검사. 최초 방문일 경우 0번 사진을 `first` 폴더에 별도 백업하여 '비포(Before)' 사진으로 영구 노출.
